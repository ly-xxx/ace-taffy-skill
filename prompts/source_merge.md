# Source Merge

当新增微博/B站材料进入时，按下面顺序处理：

1. 先判断是否是一手官方来源。
2. 再判断材料更适合进入哪一个桶：
   - `profile`
   - `distillation`
   - `expression-dna`
   - `transcripts`
3. 只追加增量，不覆盖已经稳定、且来源更强的结论。

## 合并规则

### 微博短文本

- 优先提炼：
  - 自称
  - 句尾
  - 情绪起手
  - 粉丝称呼
  - 预告/抽奖/抱怨/营业文案模板

### B站标题与简介

- 优先提炼：
  - 标题句型
  - 内容选题偏好
  - 偶像 / 游戏双重定位

### 视频/回放转录

- 优先提炼：
  - 真实说话节奏
  - 情绪跳变
  - 常见停顿和加速方式
  - 直播语境下的反应模式

## 遇到冲突时

- 官方原帖 > 二创转述
- 时间更近但来源弱，不自动覆盖时间更早但来源强的结论
- 如果无法判断，保留两种说法并标记“待人工复核”
